## 🧼 Troubleshooting

If the app does not start:

- Check that the download finished
- Make sure you opened the correct Windows file
- Try running it again as admin
- Move the app to a folder you can edit, such as your Desktop
- Re-download the file if it looks broken

If email login fails:

- Check your username and password
- Confirm that app passwords or API access are enabled
- Make sure your email provider allows third-party access
- Try signing in again from the setup screen

If mail does not appear:

- Confirm the inbox folder is correct
- Check that the app still has access
- Refresh the inbox view
- Make sure the account has new messages
